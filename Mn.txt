Я предполагаю, что вы уже создали проект C ++ Win32, в который хотите включить SQLite.

Перейдите на https://www.sqlite.org/download.html и загрузите последнюю amalgamationверсию исходного кода SQLite.
Извлеките все файлы в каталог проекта, или в свой путь включения, или в отдельный путь, который вы добавите / добавите в качестве пути включения в свойствах проекта.
Run, Developer Command Prompt for VS ****который обычно доступен по адресу Start -> Programs -> Visual Studio **** -> Visual Studio Tools.
Перейдите с помощью командной строки в тот каталог, в который мы извлекли наш SQLite.
Выполните следующую команду для компиляции: cl /c /EHsc sqlite3.c
Выполните следующую команду, чтобы создать статическую библиотеку: lib sqlite3.obj
Откройте свойства вашего проекта и добавьте sqlite3.libв Linker -> Input -> Additional Dependencies.
Теперь вы готовы к включению и использованию sqlite3.hв своем проекте.
